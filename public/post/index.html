Bar Plot

Posted on 2017.12.14 · Tagged in bar plot

Hello all! Let’s make a bar plot to show team homerun totals for every MLB team in 1980. First, install the following packages in R-Studio. library(Lahman) library(sqldf) library(devtools) library(ggplot2) Now that the packages are installed, lets extract the data. query<-"SELECT name,HR FROM Teams WHERE yearID=1980 ORDER BY HR " result<-sqldf(query) result$name<-factor(result$name,levels=result$name) After that is done, it is time to visualize the data with a bar plot. ggplot()+ geom_bar(data=result,aes(x=name,y=HR),stat='identity', color='blue',fill='black')+ coord_flip()+ xlab("Team Name")+ ylab("Homeruns")+ ggtitle("1980 Team Homerun Distribution") There you have it!…
